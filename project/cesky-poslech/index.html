Český poslech

Učím se mluvit česky, a tak jsem si sestavil aplikaci, která mi pomůže se studiem slovní zásoby. Zkoušel jsem aplikace na kartičky, ale žádná z nich ta slova neřekla. Tak jsem si vyrobil jednu, která to umí. Bylo to poprvé, co jsem pracoval s webovým řečí API. Použil jsem Preact pro UI a dozvěděl se o tom, jak používat API. Nyní mohu poslouchat slova, jak jsem si je zapamatovat.
