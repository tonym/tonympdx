CHOP eResearch

Children's Hospital of Philadelphia wanted an upgrade to their clinical trial administration UI. The existing application was outdated, difficult to use, and visually inconsistent. I worked in collaboration with the CHOP designers to create some designs using Material Design and some client requested feature upgrades. Overall, the UI became consistent, the UX improved, and everything included the then new brand identity.
